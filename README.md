## Quick Start
```bash
git clone https://github.com/milak7888/L28-Coin.git
cd L28-Coin
pip install -r requirements.txt
```

## Create Wallet
```python
from leap28.wallet.l28_wallet import L28Wallet
wallet = L28Wallet()
info = wallet.create_wallet("my_wallet")
print(info["address"])
```
